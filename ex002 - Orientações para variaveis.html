Variáveis

Variáveis são espaços na memória que guardam dados que são enviados através do usuário.

Identificadores de variáveis
  • Podem começar com letra, $ ou _
  • Não podem começar com números.
  • É possível usar letras e números.
  • Possível usar letras e símbolos.
  • Não pode conter espaço.
  • Não podem ser palavras reservadas.
    • Tipos primitivos
  • number(Para números )
    • Infinity
    • NaN
  • string(Cadeia de texto)
  • boolean(True e False)
  • null
  • undefined
  • object
    • Array
  • function
